# 🏥 شفاء كير - نظام الذكاء الاصطناعي للرعاية الصحية

## 📋 نظرة عامة

شفاء كير هو نظام متكامل لإدارة مراكز العلاج من الإدمان باستخدام الذكاء الاصطناعي. يوفر النظام إدارة شاملة للمرضى والجلسات والغرف والمصاريف مع دعم كامل للغة العربية المصرية.

## ✨ المميزات

### 🤖 الذكاء الاصطناعي
- **جلسات AI متقدمة** مع تحليل المشاعر
- **استجابة باللغة المصرية** فقط
- **تحليل سلوكي** للمرضى
- **توصيات علاجية** ذكية

### 👥 إدارة المرضى
- **ملفات مرضى شاملة** مع التاريخ الطبي
- **جدولة الجلسات** تلقائياً
- **متابعة التقدم** العلاجي
- **تقارير مفصلة** لكل مريض

### 🏠 إدارة المرافق
- **إدارة الغرف والأسرّة** مع الحجز التلقائي
- **نظام المدفوعات** المتكامل
- **إدارة المصاريف** الشخصية والمركزية
- **تقارير مالية** مفصلة

### 📊 لوحة التحكم
- **إحصائيات حية** للمركز
- **تقارير يومية** وأسبوعية وشهرية
- **مؤشرات الأداء** الرئيسية
- **تنبيهات ذكية** للأحداث المهمة

## 🚀 النشر السريع

### المتطلبات
- Node.js 22+ (مطلوب)
- حساب Supabase (مجاني)
- مفتاح Google AI (مجاني)

### خطوات النشر
1. **إنشاء قاعدة بيانات Supabase**
   ```bash
   # اتبع دليل النشر المفصل
   # DEPLOY_SUPABASE_GUIDE.md
   ```

2. **رفع الكود على GitHub**
   ```bash
   git clone https://github.com/your-username/shifa-care-ai-insights.git
   cd shifa-care-ai-insights
   ```

3. **النشر على Vercel**
   - اربط المستودع مع Vercel
   - أضف متغيرات البيئة
   - اضغط Deploy

## 🛠️ التطوير المحلي

### تثبيت المتطلبات
```bash
# تأكد من Node.js 22+
node --version
# يجب أن يظهر: v22.x.x

# تثبيت التبعيات
npm install
```

### تشغيل التطوير
```bash
# تشغيل خادم التطوير
npm run dev

# فتح المتصفح
# http://localhost:3000
```

### البناء للإنتاج
```bash
# بناء المشروع
npm run build

# معاينة البناء
npm run preview
```

## 🔧 الإعدادات

### متغيرات البيئة
```
# Supabase
VITE_SUPABASE_URL=your-supabase-url
VITE_SUPABASE_ANON_KEY=your-anon-key

# Google AI
VITE_GOOGLE_AI_API_KEY=your-google-ai-api-key
VITE_GOOGLE_AI_MODEL=gemini-1.5-flash

# التطبيق
VITE_APP_NAME=شفاء كير
VITE_APP_VERSION=2025.2.1
VITE_AI_RESPONSE_LANGUAGE=egyptian_arabic
```
