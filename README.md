# 🧠 شفا كير - نظام إدارة مراكز علاج الإدمان مع الذكاء الاصطناعي

## 📋 نظرة عامة

شفا كير هو نظام متكامل لإدارة مراكز علاج الإدمان، مزود بخدمات ذكاء اصطناعي متقدمة لتحليل الجلسات العلاجية وتقديم رؤى ذكية للمعالجين والمرضى.

## 🚀 التثبيت والإعداد

### المتطلبات الأساسية
- Node.js 18+ 
- npm أو yarn
- حساب Supabase
- حساب OpenAI (للخدمات الذكية)

### خطوات التثبيت

1. **استنساخ المشروع**
```bash
git clone https://github.com/your-username/shifa-care-ai-insights.git
cd shifa-care-ai-insights
```

2. **تثبيت التبعيات**
```bash
npm install
```

3. **إعداد متغيرات البيئة**
```bash
cp env.example .env.local
```

4. **تكوين Supabase**
- أنشئ مشروع في [Supabase](https://supabase.com)
- انسخ URL والمفتاح العام
- أضفهما في `.env.local`:
```bash
VITE_SUPABASE_URL=your_supabase_project_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
```

5. **إعداد OpenAI (اختياري ولكن مُوصى به)**
- اذهب إلى [platform.openai.com](https://platform.openai.com)
- أنشئ حساب واحصل على مفتاح API
- أضفه في `.env.local`:
```bash
VITE_OPENAI_API_KEY=your_openai_api_key
```

6. **تشغيل التطبيق**
```bash
npm run dev
```

## 🧠 إعداد الذكاء الاصطناعي

### الحصول على مفتاح OpenAI

1. اذهب إلى [platform.openai.com](https://platform.openai.com)
2. سجل دخول أو أنشئ حساب جديد
3. اذهب إلى "API Keys"
4. انقر على "Create new secret key"
5. انسخ المفتاح وضعه في `VITE_OPENAI_API_KEY`

### اختبار الإعداد

1. شغل التطبيق: `npm run dev`
2. اذهب إلى `/admin`
3. انقر على تبويب "الذكاء الاصطناعي"
4. انقر على "اختبار الاتصال"

### الميزات المتاحة بعد الإعداد

- ✅ تحليل الجلسات العلاجية
- ✅ اقتراح خطط علاجية مخصصة
- ✅ تقييم مخاطر الانتكاس
- ✅ إنشاء تقارير ذكية
- ✅ اقتراح أنشطة علاجية
